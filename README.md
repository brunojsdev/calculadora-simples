# Calculadora Simples

Projeto de uma calculadora simples desenvolvida exclusivamente com **HTML**, **CSS** e **JavaScript**, com os arquivos organizados separadamente por linguagem.

## Descrição

A aplicação implementa uma calculadora funcional para operações matemáticas básicas, executada diretamente no navegador, sem dependências externas. A interface é responsiva e utiliza eventos de clique e teclado para interação.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão  
- Suporte a números decimais  
- Porcentagem  
- Limpar cálculo (C)  
- Apagar último caractere (←)  
- Entrada por teclado  
- Tratamento básico de erros matemáticos  

## Estrutura de Arquivos

- `index.html` – Estrutura da interface da calculadora  
- `styles.css` – Estilização visual e layout  
- `script.js` – Lógica de funcionamento e manipulação de eventos  

## Como Executar

1. Coloque os arquivos `index.html`, `styles.css` e `script.js` na mesma pasta  
2. Abra o arquivo `index.html` em qualquer navegador moderno  

## Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  

## Ambiente

A aplicação roda inteiramente no lado do cliente e não requer servidor ou bibliotecas adicionais.
